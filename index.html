About Me

My Programming and Development Journey

Hello! I'm GM. Pawan, a passionate student currently in the 12th standard, and an aspiring software developer. I have embarked on a journey to master various programming languages and technologies. My learning experience spans from foundational skills to advanced AI and game development, reflecting my commitment to becoming a versatile and skilled programmer. Below is an overview of my expertise.

Scratch Programming

My programming journey began with Scratch, a visual programming language that introduced me to the basics of coding and logical thinking. Through creating interactive stories and games, I developed a strong foundation in coding principles.

Python Programming

Python is one of the first text-based programming languages I mastered, offering a deep dive into data structures, algorithms, and problem-solving techniques. I have used Python extensively for software development, data analysis, and exploring machine learning concepts.

Flutter & Dart Development

With a keen interest in mobile app development, I learned Flutter and Dart to create cross-platform applications with beautiful UIs. This skill set has enabled me to develop responsive and efficient mobile apps that work seamlessly on both Android and iOS devices.

TensorFlow & Advanced AI Expertise

I have ventured into the world of Artificial Intelligence (AI) with TensorFlow, gaining hands-on experience in machine learning models, neural networks, and deep learning techniques. This knowledge allows me to build smart applications capable of complex decision-making and data analysis.

Microsoft Certification in AI & Cloud Technologies

To further enhance my expertise, I obtained Microsoft certifications in AI and cloud technologies. These certifications validate my skills in designing AI solutions and utilizing cloud platforms like Azure for scalable and secure application development.

Web Development Fundamentals

My web development skills encompass HTML, CSS, and JavaScript, providing a solid foundation for building dynamic and interactive websites. I have created web applications that focus on user experience, responsive design, and modern web standards.

Game Development Experience with Unity and Roblox

Game development is a significant part of my learning journey, with experience in both Unity and Roblox platforms. I have developed multiple games, honing skills in C# scripting, 3D modeling, Lua scripting, and multiplayer game design, blending creativity with technical proficiency.
